# 🏷️ Sistema de Leilões

Sistema desenvolvido para gerenciar produtos em uma casa de leilões, permitindo cadastro e listagem de itens.

## 🚀 Tecnologias utilizadas
- Java
- MySQL
- Git e GitHub

## 📌 Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Integração com banco de dados

## 💡 Aprendizados
- Conexão com banco de dados (MySQL)
- Organização de projeto em Java
- Uso de versionamento com Git

## 📊 Possíveis melhorias
- Interface gráfica (GUI)
- Sistema de lances em tempo real
- Autenticação de usuários

## 📁 Estrutura do projeto
- `src/` → Código fonte
- `nbproject/` → Configurações do projeto
- `build.xml` → Build do projeto
